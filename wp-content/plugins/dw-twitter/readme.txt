== Description ==

This is a simple plugin that collect and display your recent tweets using search queries: from, @ or #. The plugin also supports Twitter API 1.1.

== Installation ==

1. Log in to Twitter Developer account, then create a new application on: https://dev.twitter.com/apps/new. This step, you will get Consumer Key & Consumer Secret at OAuth settings.
2. Upload the plugin directory to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Drag & Drop "DW Twitter" widget to your Sidebar through the 'Widgets' menu in WordPress
5. Input your Twitter app's Consumer Key & Consumer Secret from Step 1.
6. Check in the box if you want show follow button, account info and user avatar
